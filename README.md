c5block
=======

A Theme for Novius OS which allows to create concrete5 block template files.

## Requirements

* Novius OS Elche

## Bundled Library

* SimpleJSXslt.js
* https://github.com/shimohi/SimpleJSXslt

* jQuery
* http://jquery.com/

## Example
See [here](https://www.rescuework.jp/concrete5_db_xml.html)
